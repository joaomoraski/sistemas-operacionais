# Como executar os códigos.

### Primeiramente é necessário executar o comando 
```
make compile
```

###### Exercício 01
Para executar o código digite no terminal.<br>
A quantidade de parametros é opcional. pois se ele não receber os N o programa solicita antes de rodar.
```
make run1

OR

./ex01-hierarquia-processos <Numero>
```

###### Exercício 02
Para executar o código digite no terminal<br>
No comando abaixo ele ira executar diretamente o comando ls
```
make run2

OR

./ex02-comando-processo <comando>
```

###### Exercício 03
Para executar o código digite no terminal<br>
Já possui os parametros padrões, porem caso seja necessario a ordem esta a baixo
```
make run3

OR

./ex03-divide-vetor <Numero filho> <Valor a ser procurado>
```

###### Exercício 04
Para executar o código digite no terminal<br>
O terminal não possui valores na hora de sua execução.
```
make run4
```
